## 爬虫

- [xs00](https://github.com/bitmonk404/500lines/blob/master/crawler/xs00.py)
不可能世界爬虫，简易版:  
1、Tornado实现异步爬取数据。使用队列的方式控制并发数量。  
2、pyquery解析dom树。  
3、使用TinyDB存储数据。  
- [xs01](https://github.com/bitmonk404/500lines/blob/master/crawler/xs01.py)
增加功能：  
1、逆向分析，ajax动态数据爬取（评论部分）。  
2、爬取小说内容。  
3、使用Mongodb数据库。
